# Guess the Flag

An iOS game that tests your knowledge of country flags. Guess which flag matches the displayed country name.

## Description

The project is a flag-guessing game: the user sees a country name and three flag buttons. After selecting one, the app shows whether the answer was correct or wrong, updates the score, and proceeds to the next round. The game ends after 8 questions (or 3 for testing) and displays a final alert with the total score and a restart option.

## Features

- Three randomly selected flags per round
- Score tracking (+1 for correct, –1 for wrong)
- Alert after each round showing result and current score
- Game limit (configurable number of questions)
- Final alert with total score and restart button
- Radial gradient background and semi-transparent card for flags

## Tech Stack

- **Swift 5**
- **SwiftUI**
- **Xcode 15+**
- **iOS 18.0+**
